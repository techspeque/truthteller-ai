# TruthTeller AI v0.3.0

Released: 2026-03-29

## Features

- Added model status badges so the UI surfaces per-model progress and failure states more clearly during council runs.
- Switched council insights from inline sub-panels to a dialog-based presentation with broader failure-state test coverage.
- Expanded product documentation with new getting started, architecture, development, and testing/release guides.

## Fixes

- Corrected the insights panel opacity so the modal presentation remains readable.
- Tightened stage/event plumbing across the frontend, server, and Tauri adapter to support the new model status display reliably.

## Chore

- Added tagged release automation that verifies version alignment, checks for matching `docs/releases/vX.Y.Z.md` metadata, renders the GitHub release body, and publishes macOS, Linux, and Windows artifacts.
- Refreshed packaged application icons and related release assets.
